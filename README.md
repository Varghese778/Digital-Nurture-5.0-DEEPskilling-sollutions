# Digital Nurture 5.0 — Java FSE DEEPskilling Solutions

Hands-on training and practice solutions for the **Cognizant Digital Nurture 5.0** program, Java Full-Stack Engineering (FSE) track.

This repository contains exercise question papers, weekly hands-on solutions, and reference materials covering the complete 7-week curriculum.

---

## Repository Structure

The project is structured into curriculum resource directories, weekly solutions, and helper tools:

